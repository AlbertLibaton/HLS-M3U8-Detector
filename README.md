# HLS-M3U8-Detector
HLS / M3U8 Detector is a Chrome extension designed for developers, testers, and streaming enthusiasts. It monitors network requests and identifies HLS playlist files (.m3u8) used by modern video streaming platforms.  The extension helps users quickly discover HLS stream URLs for debugging, development, quality assurance, and educational purposes.


# How It Works
Open a webpage containing a video player.
The extension monitors network traffic.
When an HLS playlist (.m3u8) is detected, it is displayed instantly.
Copy the detected URL for analysis or testing.
# Use Cases
Video player debugging
Streaming application development
HLS workflow testing
Network traffic analysis
Educational and research purposes
# Privacy

This extension processes network requests locally within the browser and does not collect, store, or transmit personal data.
